# HelloWorld kata

Inspired from [CodingDojo.org](https://codingdojo.org/kata/Hello/)

## Goal
Output the message "Hello, World!" in console.

## Bonus
Add the option to have the message surrounded by "*" like so
```
***************
*Hello, World!*
***************
```
